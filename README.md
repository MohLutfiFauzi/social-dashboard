## Run Locally

1. **Clone the repository**
   git clone <https://github.com/mohlutfifauzi/social-dashboard.git>
   cd social-dashboard

2. **Install dependencies**
   npm install (using node : 20.13.0)

3. **Start the development server**
   npm start

4. **Open in browser**
   <http://localhost:5174/users>
